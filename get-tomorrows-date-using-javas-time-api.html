Get Tomorrow's Date Using Java's Time API

Java Time API Quickstart
Previous
Next

In order to get tomorrow's date, first find the current date, and then add one day to it.

The plus method will return a copy of the date with the specified amount added.

import java.time.LocalDate;
import java.time.Period;

public final class DateUtility {

	private DateUtility() {}

	public static LocalDate getTomorrow() {
		return LocalDate.now().plus(Period.ofDays(1));
	}

}
