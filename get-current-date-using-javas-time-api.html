Get Current Date Using Java's Time API

Java Time API Quickstart
Next

Use the LocalDate.now() method to obtain the current date from the system clock in the default time-zone.

The plus method will return a copy of the date with the specified amount added. 

import java.time.LocalDate;
                                                                
public final class DateUtility {
    
    private DateUtility() {}
    
    public static LocalDate getCurrentDate() {
        return LocalDate.now();
    }
    
}
